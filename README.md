#  Abstract (EN)

This application provides a practical solution for managing and controlling access to restricted or critical areas, such as sensitive material storage facilities, strategic infrastructure, laboratories, and data centers. Its primary goal is to reduce risks associated with unauthorized physical access, which may threaten physical security, information confidentiality, and organizational continuity.

The system features an intuitive and adaptable interface designed for various user roles, including applicants, security personnel, supervisors, and administrators. It streamlines the processes of access request submission, validation, tracking, and documentation by integrating user authentication, permission verification, automated event logging, and customizable reporting tools.

The workflow begins with the digital registration of the applicant, followed by the submission of access requests through a form that captures essential details such as schedule, personnel involved, and justification. Each request is evaluated by the designated area supervisor based on the organizations established security policies.

All operations are stored in a centralized database, ensuring complete traceability and serving as a valuable resource for internal and external audits.

Overall, the system presents a simple yet strategic tool for strengthening physical and administrative security in sensitive environments.
